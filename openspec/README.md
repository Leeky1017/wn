# OpenSpec（WriteNow）

OpenSpec 用于记录可验证的需求增量（spec-first）与运行证据（task runs）。

- Specs：`openspec/specs/**/spec.md`
- Task runs：`openspec/_ops/task_runs/ISSUE-N.md`

## Official layout (CLI)

- Project: `openspec/project.md`
- Handoff: `openspec/AGENTS.md`
- Changes: `openspec/changes/` (archive: `openspec/changes/archive/`)

## Canonical specs (WriteNow)

- Constitution：`openspec/specs/writenow-constitution/spec.md`
- Delivery workflow：`openspec/specs/writenow-delivery-workflow/spec.md`
- Spec writing standard：`openspec/specs/openspec-writing-standard/spec.md`
